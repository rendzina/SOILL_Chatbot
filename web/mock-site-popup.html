EU Mission Soil Living Labs

This page simulates a separate project website with a floating chat button in the bottom-right corner (Approach 1).

Try it

Click the chat button at the bottom right of this page. The panel loads the same chat UI in an iframe.

Press Esc or click the button again to close the panel.

Embed on your project site

Include the stylesheet and widget script on any page. Set data-chat-url to your deployed chat page URL. 

<link rel="stylesheet" href="https://your-api-host/web/mock-site.css">
<script
  src="https://your-api-host/web/widget-iframe.js"
  data-chat-url="https://your-api-host/web/">
</script>

For a native widget (no iframe) that calls POST /api/chat directly from your site, configure CORS on the API — see documents/deployment.md.
