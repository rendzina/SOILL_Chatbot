This page simulates a separate project website. The chatbot is integrated as a dedicated page rather than a floating widget.

Option A — Link to chat page

Add a navigation item or button that opens the chat UI in the same tab or a new tab.

Ask SOILL Open in new tab

In production, the link would point to your deployed API host, e.g. https://chat-api.example.org/web/.

Option B — Embed chat in a page section

Embed the chat UI in an iframe on a route such as /ask-soill on your project site.

Example embed code for your CMS or static site: 

<iframe
  src="https://your-api-host/web/"
  title="SOILL chatbot"
  style="width:100%;height:70vh;border:none;border-radius:12px;">
</iframe>
